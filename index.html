The web is made up of structured documents written in a form computers can understand called HTML

The Web includes HTML documents, as well as music, videos, images, and other files that are stored on servers and read by our web browser via a protocol called HTTP.

HTML include the text that will be visible, plus the markup that makes that text appear certain ways.

There is also another language called CSS that allows further customization of the appearance of web pages and we will learn about it later in the NanoDegree.

HTML includes tags that contain content as well as void tags
that do not contain content. Tags can be either inline such as span or img which means they appear inline like this

or they can be block such as the p tag which creates an invisible block around the element

There are also container tags like div that allow us to add attributes and style certain areas of a document.
